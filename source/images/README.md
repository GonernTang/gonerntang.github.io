# Images Directory

请将以下图片文件放置在此目录中：

## 必需的图片文件

1. **avatar.gif** - 博客头像图片
   - 建议尺寸: 200x200 像素
   - 格式: gif, jpg, png
   - 位置: `/source/images/avatar.gif`

2. **favicon-16x16.png** - 16x16 图标
   - 尺寸: 16x16 像素
   - 位置: `/source/images/favicon-16x16.png`

3. **favicon-32x32.png** - 32x32 图标
   - 尺寸: 32x32 像素
   - 位置: `/source/images/favicon-32x32.png`

4. **apple-touch-icon.png** - Apple 触摸图标
   - 建议尺寸: 180x180 像素
   - 位置: `/source/images/apple-touch-icon.png`

5. **safari-pinned-tab.svg** - Safari 固定标签图标
   - 格式: SVG
   - 位置: `/source/images/safari-pinned-tab.svg`

## 在线工具

- Favicon 生成器: https://realfavicongenerator.net/
- 图片压缩: https://tinypng.com/
- 图片裁剪: https://www.iloveimg.com/crop-image

## 注意事项

- 图片文件名必须与上述名称完全一致
- 建议使用优化后的图片以加快网站加载速度
